Exemplo de Destructuring com Objetos

O destructuring é uma técnica poderosa no JavaScript que permite desestruturar objetos e arrays, facilitando a extração de valores diretamente para variáveis. Neste exemplo, utilizamos destructuring dentro de uma função para acessar propriedades específicas de um objeto. A função teste recebe um objeto como parâmetro e, através da sintaxe { a: x, d, z = 10 }, extrai as propriedades a, d, e atribui um valor padrão para z caso não esteja presente no objeto. Ao chamar a função, as propriedades extraídas são exibidas no console.
