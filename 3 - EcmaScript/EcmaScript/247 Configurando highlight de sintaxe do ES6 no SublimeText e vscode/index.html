Configurando highlight de sintaxe do ES6 no SublimeText

Para configurar o destaque de sintaxe do ES6 no SublimeText, siga os passos abaixo:

  1. Acesse o menu Preferences > Package Control.
  2. No campo de busca, procure por "Babel" e selecione a opção correspondente.
  3. Após a instalação, acesse o menu View > Syntax > Open all with current extension as... e selecione Babel > JavaScript.
  4. Agora, seu SublimeText estará configurado para destacar a sintaxe ES6 de forma apropriada.


247 Configurando highlight de sintaxe do ES6 no VSCode

Para configurar o destaque de sintaxe do ES6 no Visual Studio Code (VSCode), siga os passos abaixo:

  1. Acesse a aba Extensões clicando no ícone do lado esquerdo ou pressionando Ctrl+Shift+X.
  2. Pesquise por "ES6 Syntax" ou "Babel JavaScript" e instale a extensão sugerida.
  3. Certifique-se de que a linguagem do arquivo está configurada como JavaScript clicando no canto inferior direito e selecionando JavaScript.
  4. Com isso, o destaque de sintaxe ES6 será aplicado automaticamente em seus arquivos .js.
