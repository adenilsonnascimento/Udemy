Operador Spread no JavaScript

O operador spread (...) no JavaScript permite "espalhar" os elementos de um array, string ou objeto em um novo contexto, como outro array ou objeto.

Em strings: Cada caractere pode ser "espalhado" individualmente, facilitando sua manipulação ou conversão em arrays.

Em arrays: Permite combinar elementos de vários arrays em um único array ou adicionar elementos adicionais ao início, meio ou fim de outro array.

Em objetos: O operador é útil para clonar objetos, além de possibilitar a adição ou modificação de propriedades sem alterar o objeto original.

O operador spread simplifica a manipulação de dados e ajuda a evitar cópias profundas manuais, promovendo um código mais limpo e conciso.
