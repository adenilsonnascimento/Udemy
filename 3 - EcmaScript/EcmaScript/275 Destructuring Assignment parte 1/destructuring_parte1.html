Destructuring no JavaScript

O destructuring é uma sintaxe no JavaScript que permite a extração de valores de arrays ou propriedades de objetos e atribuí-los a variáveis de forma mais compacta.

Explicação do código:

  • No primeiro exemplo, usamos destructuring para pegar os elementos de um array de frutas e atribuí-los a variáveis específicas. Caso não haja valor para uma posição, um valor padrão pode ser atribuído à variável.
  • No segundo exemplo, o destructuring é usado de forma aninhada para acessar elementos dentro de arrays dentro de um array, extraindo o nome 'Maria' da segunda posição do segundo sub-array.
