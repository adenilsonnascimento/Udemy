Modificando a classe do elemento

Explicações dos códigos comentados

O código tem como objetivo modificar a classe CSS de um elemento HTML, alterando seu estilo de apresentação. Inicialmente, o elemento `

` possui a classe `estilo1`, que define a cor de fundo como vermelha e o texto como branco. Quando o usuário clica no botão, a função `modificaClasse()` é acionada, e a classe do elemento com o ID `mensagem` é alterada para `estilo2`, que aplica um estilo diferente: uma largura de 400px, altura de 50px, texto centralizado, tamanho de fonte maior (25px) e um padding de 25px. Isso faz com que a aparência do elemento mude ao ser clicado no botão.
