Explicando tudo sobre o BOM location

O **BOM (Browser Object Model)** é um conjunto de objetos que permite o acesso e manipulação de funcionalidades do navegador. Um desses objetos é o **`window.location`**, que fornece informações sobre a URL atual e permite navegar para outras páginas.

Neste exemplo, os botões utilizam o método window.location.href, que altera o endereço da página atual para o URL especificado. Quando o botão "Acessar Google" é clicado, o navegador redireciona para o site do Google. Da mesma forma, ao clicar em "Acessar Faus", o navegador redireciona para o site da Faus.

Explicando todos códigos existentes no BOM location

O objeto **`window.location`** oferece várias propriedades e métodos que podem ser usados para interagir com URLs. Aqui estão os principais:

  • location.href: Retorna a URL atual ou redireciona para uma nova URL.
  • location.hostname: Retorna o nome do host (ex: www.google.com).
  • location.pathname: Retorna o caminho do arquivo da URL atual (ex: /pagina.html).
  • location.protocol: Retorna o protocolo da URL (ex: http ou https).
  • location.assign(): Carrega um novo documento em uma URL especificada.
  • location.reload(): Recarrega a página atual.
Esses comandos permitem desde navegação até recarregamento de páginas, tornando o **BOM location** uma ferramenta poderosa para interações com o navegador.
