Explicações dos códigos comentados

O código acima utiliza alguns dos métodos mais comuns do DOM para selecionar e manipular elementos HTML. Vamos entender o que está acontecendo:

1. Seleção pelo ID:

O método document.getElementById('nome_usuario') seleciona o elemento com o ID 'nome_usuario'. Ele retorna um único elemento, já que IDs devem ser únicos em uma página. Esse método é muito útil quando queremos acessar um único elemento de forma eficiente.

2. Seleção por Tag Name:

O método document.getElementsByTagName('input') seleciona todos os elementos do tipo <input> na página. Ele retorna uma coleção de todos os inputs, o que pode ser útil quando precisamos acessar múltiplos elementos de uma vez.

3. Seleção por Class Name:

O método document.getElementsByClassName('campo_texto') seleciona todos os elementos com a classe 'campo_texto'. Esse método retorna uma coleção de todos os elementos que compartilham essa classe, permitindo modificar vários elementos ao mesmo tempo.

4. Seleção por Name:

O método document.getElementsByName('nome') seleciona todos os elementos que têm o atributo name="nome". Esse método pode ser útil para selecionar elementos de formulários, como campos de texto, com o mesmo nome.
