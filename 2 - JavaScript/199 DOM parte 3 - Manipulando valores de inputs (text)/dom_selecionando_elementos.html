Letras: Números:

O código permite que o usuário digite um caractere no campo de entrada. A função `distribuiCaracter()` é chamada sempre que uma tecla é pressionada. O valor digitado é capturado e, em seguida, o campo de entrada é limpo. A função remove espaços em branco nas extremidades do texto utilizando o método `.trim()`. Em seguida, o código verifica se o caractere digitado é um número (de `0` a `9`). Se for, ele é inserido no campo "apenas_numeros", caso contrário, é tratado como uma letra e inserido no campo "apenas_letras". Esses campos são desabilitados para que o usuário não possa editar diretamente os resultados, sendo apenas para visualização do que foi categorizado.
