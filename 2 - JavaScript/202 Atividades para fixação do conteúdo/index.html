Explicação do código

Este código manipula o estilo de fundo de um campo de texto (``) com base em dois eventos: `onfocus` e `onblur`. Quando o campo de texto recebe o foco (quando o usuário clica ou navega até o campo), a função `acionaCampo()` é chamada, mudando a cor de fundo do campo para amarela. Quando o campo perde o foco (quando o usuário sai do campo), a função `validaCampo()` é chamada. Esta função verifica o comprimento do texto digitado no campo. Se o texto for menor que 3 caracteres, o fundo do campo é alterado para vermelho. Caso contrário, a cor de fundo do campo muda para verde.
