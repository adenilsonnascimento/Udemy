Exibição da Data Atual

Este script utiliza o objeto Date do JavaScript para obter a data atual e exibi-la no formato DD/MM/AAAA. O objeto Date representa um momento no tempo e oferece diversos métodos para manipular e acessar componentes individuais da data, como o dia, o mês e o ano.

O método getDate() retorna o dia do mês (de 1 a 31). O método getMonth() retorna o índice do mês (de 0 a 11), sendo necessário somar 1 para obter o número correto do mês. Já o método getFullYear() retorna o ano com quatro dígitos. O código combina esses componentes da data em uma string no formato desejado.

Para exibir a data no HTML, foi adicionado um elemento <span> com o atributo id="data-atual". Este elemento é usado como alvo para inserir a data gerada pelo script. O código acessa este elemento utilizando document.getElementById() e atualiza seu conteúdo com a data formatada.

Além disso, foi utilizado o evento DOMContentLoaded para garantir que o script só será executado após todos os elementos do DOM estarem carregados. Isso evita erros que poderiam ocorrer se o JavaScript fosse executado antes do carregamento do elemento <span>.
