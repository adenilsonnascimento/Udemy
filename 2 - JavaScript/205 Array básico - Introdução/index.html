Explicando Arrays

Um array é uma estrutura de dados que permite armazenar vários valores em uma única variável. Em vez de declarar várias variáveis para armazenar dados relacionados, você pode usar um array.

Exemplos de arrays e como utilizá-los

Veja abaixo alguns exemplos de como criar e acessar arrays em JavaScript.

Exemplo 1: Array de números

Um array pode armazenar números. Veja o exemplo: 

        // Criando um array de números
        let numeros = [10, 20, 30, 40, 50];
        
        // Acessando um valor específico no array
        console.log(numeros[0]); // Exibe o primeiro número do array (10)
        console.log(numeros[3]); // Exibe o quarto número do array (40)

Exemplo 2: Array de strings

Arrays também podem armazenar strings. Exemplo: 

        // Criando um array de strings
        let frutas = ["maçã", "banana", "laranja", "uva"];
        
        // Acessando um valor específico no array
        console.log(frutas[1]); // Exibe "banana"
        console.log(frutas[2]); // Exibe "laranja"

Exemplo 3: Array misto (números e strings)

Arrays também podem conter diferentes tipos de dados. Veja um exemplo com números e strings misturados: 

        // Criando um array com números e strings
        let misto = [25, "João", 12, "Maria"];
        
        // Acessando valores
        console.log(misto[0]); // Exibe 25
        console.log(misto[1]); // Exibe "João"

Exemplo 4: Alterando elementos de um array

É possível alterar o valor de um elemento de um array. Exemplo: 

        let carros = ["Fusca", "Brasília", "Chevette"];
        
        // Alterando o primeiro elemento
        carros[0] = "Civic";
        console.log(carros[0]); // Exibe "Civic"

Exemplo 5: Adicionando elementos a um array

Podemos adicionar elementos a um array usando o método push(). Veja o exemplo: 

        let cores = ["azul", "verde", "vermelho"];
        
        // Adicionando um novo elemento no final
        cores.push("amarelo");
        console.log(cores); // Exibe ["azul", "verde", "vermelho", "amarelo"]

Exemplo 6: Percorrendo um array com laços (loops)

Você pode percorrer um array usando loops, como o for. Exemplo: 

        let animais = ["gato", "cachorro", "pássaro", "peixe"];
        
        // Usando um loop for para percorrer o array
        for (let i = 0; i < animais.length; i++) {
            console.log(animais[i]);
        }
