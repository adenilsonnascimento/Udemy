Este código demonstra como criar um array em JavaScript, adicionar elementos a ele e percorrê-lo utilizando uma estrutura de repetição while.

Passo a passo do código:
- Um array vazio chamado lista_frutas é criado utilizando a função Array().
- Elementos como "Banana", "Maçã", "Morango" e "Uva" são adicionados ao array em posições específicas (índices 0 a 3).
- Uma variável chamada y é inicializada com o valor 0 para ser usada como contador.
- Um laço while percorre o array lista_frutas, exibindo cada elemento no documento. O loop continua enquanto y for menor que o comprimento do array (obtido com lista_frutas.length).
- A cada iteração, o valor de y é incrementado em 1 para que o próximo elemento seja acessado na próxima execução do loop.
- O comando document.write é utilizado para imprimir os elementos do array, seguidos de uma quebra de linha.

Resultado:
- O navegador exibirá os elementos "Banana", "Maçã", "Morango" e "Uva", cada um em uma linha separada.
