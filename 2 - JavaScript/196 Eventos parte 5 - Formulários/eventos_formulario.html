Explicações dos códigos comentados

No código acima, temos dois tipos de eventos: onfocus, onblur, e onchange. Cada um desses eventos é associado a diferentes elementos da interface do usuário, permitindo capturar interações específicas com o formulário.

O evento onfocus é acionado quando o campo de entrada (input) recebe foco, ou seja, quando o usuário clica ou começa a digitar dentro do campo. Quando o foco é recebido, a mensagem "recebeu o foco" é exibida no console do navegador. Já o evento onblur é disparado quando o campo perde o foco, ou seja, quando o usuário clica fora do campo ou navega para outro lugar. A mensagem "perdeu o foco" será exibida no console quando esse evento ocorrer.
