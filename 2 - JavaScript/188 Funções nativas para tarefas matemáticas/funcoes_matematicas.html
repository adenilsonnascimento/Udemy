Vamos explicar cada parte do código JavaScript, onde estão sendo usados métodos do objeto Math para realizar diferentes operações matemáticas.

1. Método Math.ceil()

var x = Math.ceil(10.380)
document.write(x)
O método Math.ceil() arredonda um número para cima, ou seja, para o próximo número inteiro mais alto.
No exemplo, 10.380 será arredondado para 11.
Saída esperada: 11

2. Método Math.floor()

var y = Math.floor(10.380)
document.write(y)
O método Math.floor() arredonda um número para baixo, ou seja, para o próximo número inteiro mais baixo.
No exemplo, 10.380 será arredondado para 10.
Saída esperada: 10

3. Método Math.round()

var z = Math.round(10.380)
document.write(z)
O método Math.round() arredonda um número para o inteiro mais próximo. Se a parte decimal for 0.5 ou maior, arredonda para cima; caso contrário, arredonda para baixo.
No exemplo, como 10.380 tem uma parte decimal menor que 0.5, será arredondado para 10.
Saída esperada: 10

4. Método Math.random()

var c = Math.random()
document.write(c)
O método Math.random() retorna um número pseudoaleatório entre 0 (inclusivo) e 1 (exclusivo), ou seja, o resultado será um número decimal entre 0 e quase 1.
A saída será um valor diferente a cada execução, e sempre será um valor decimal menor que 1.
Saída esperada: Um número aleatório entre 0 e 1, como 0.73242.

Resumo dos métodos usados:
Math.ceil(): Arredonda um número para cima, para o próximo número inteiro.
Math.floor(): Arredonda um número para baixo, para o próximo número inteiro mais baixo.
Math.round(): Arredonda um número para o inteiro mais próximo.
Math.random(): Gera um número pseudoaleatório entre 0 e 1.
