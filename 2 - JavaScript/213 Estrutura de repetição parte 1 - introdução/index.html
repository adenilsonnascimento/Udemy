O que são e para que servem as Estruturas de Repetição?

As estruturas de repetição são um conceito fundamental na programação. Elas permitem que um bloco de código seja repetido várias vezes, com base em uma condição. Isso é útil quando precisamos realizar tarefas repetitivas de forma eficiente. Existem três principais tipos de estruturas de repetição em JavaScript:

Tipos de Estruturas de Repetição e para que cada uma delas serve:

1. For: A estrutura for é usada quando sabemos de antemão quantas vezes o código precisa ser repetido. Por exemplo, quando queremos percorrer um array ou realizar uma contagem específica.

2. While: A estrutura while é utilizada quando não sabemos quantas vezes o código precisa ser repetido, mas queremos que o código continue rodando enquanto uma condição for verdadeira.

3. Do...While: A estrutura do...while é semelhante ao while, mas a principal diferença é que o código dentro do do é executado pelo menos uma vez, antes de verificar a condição do while.

Código de cada estrutura de repetição em JavaScript:

1. Estrutura for:

O laço for é utilizado para realizar uma repetição baseada em um contador.


for (let i = 0; i < 5; i++) {
    console.log('Repetição número ' + i);
}

Este código vai imprimir os números de 0 a 4, repetindo 5 vezes.

2. Estrutura while:

O laço while executa o código enquanto a condição for verdadeira.


let i = 0;
while (i < 5) {
    console.log('Repetição número ' + i);
    i++;
}

Este código também irá imprimir os números de 0 a 4, repetindo enquanto a condição i < 5 for verdadeira.

3. Estrutura do...while:

O laço do...while é executado pelo menos uma vez, independentemente da condição.


let i = 0;
do {
    console.log('Repetição número ' + i);
    i++;
} while (i < 5);

Este código também vai imprimir os números de 0 a 4, mas garantirá que o bloco de código seja executado pelo menos uma vez, mesmo que a condição i < 5 já seja falsa.
