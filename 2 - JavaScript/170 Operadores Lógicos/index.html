Os operadores lógicos em JavaScript são usados para realizar operações lógicas com variáveis ou expressões, retornando valores booleanos (true ou false). Eles são frequentemente utilizados em estruturas de controle como if, while, e em expressões condicionais.

Aqui estão os principais operadores lógicos em JavaScript:

1. AND Lógico (&&)
O operador AND retorna true se ambas as expressões forem verdadeiras. Se uma ou ambas forem falsas, ele retorna false.

Sintaxe:

javascript
Copiar código
expr1 && expr2

Exemplo:

javascript
Copiar código
let a = true;
let b = false;
console.log(a && b); // Saída: false
console.log(a && true); // Saída: true
2. OR Lógico (||)
O operador OR retorna true se pelo menos uma das expressões for verdadeira. Se ambas forem falsas, ele retorna false.

Sintaxe:

javascript
Copiar código
expr1 || expr2

Exemplo:

javascript
Copiar código
let a = true;
let b = false;
console.log(a || b); // Saída: false
console.log(b || false); // Saída: false
3. NOT Lógico (!)
O operador NOT inverte o valor de uma expressão booleana. Se for true, ele retorna false, e vice-versa.

Sintaxe:

javascript
Copiar código
!expr

Exemplo:

javascript
Copiar código
let a = true;
console.log(!a); // Saída: false
console.log(!false); // Saída: true
4. Nullish Coalescing (??)
O operador Nullish Coalescing é usado para retornar o valor à direita se o valor à esquerda for null ou undefined. Se o valor à esquerda não for nulo ou indefinido, ele retorna o valor à esquerda.

Sintaxe:

javascript
Copiar código
expr1 ?? expr2

Exemplo:

javascript
Copiar código
let a = null;
let b = "valor padrão";
console.log(a ?? b); // Saída: "valor padrão"
5. Operador Ternário (? :)
O operador ternário é uma forma curta de escrever uma condicional if...else, verificando uma expressão e retornando um dos dois valores.

Sintaxe:

javascript
Copiar código
condicao ? expr1 : expr2

Exemplo:

javascript
Copiar código
let idade = 18;
let resultado = (idade >= 18) ? "Maior de idade" : "Menor de idade";
console.log(resultado); // Saída: "Maior de idade"
Esses operadores são fundamentais para controle de fluxo e decisões em JavaScript, permitindo a criação de expressões condicionais complexas.
