Explicando o que é o BOM

O BOM (Browser Object Model) é uma interface que permite interagir com o navegador através do JavaScript. Ele fornece métodos e propriedades para manipular a janela do navegador, o histórico, os cookies, entre outros.

Como e onde usar o BOM

O BOM pode ser usado em várias situações, como redirecionar o usuário para outra página, abrir uma nova janela, verificar informações sobre o navegador ou atualizar a URL atual. É utilizado principalmente em aplicativos web que precisam interagir com o ambiente do navegador.

Códigos do BOM

Segue alguns exemplos de uso do BOM:

  • Abrir uma nova janela:
    • window.open("https://www.example.com", "_blank");
  • Exibir um alerta:
    • window.alert("Bem-vindo ao site!");
  • Redirecionar para outra página:
    • window.location.href = "https://www.example.com";
  • Verificar o tamanho da janela:
    • 
console.log("Largura: " + window.innerWidth);
console.log("Altura: " + window.innerHeight);
  • Interagir com o histórico do navegador:
    • window.history.back(); // Volta uma página
window.history.forward(); // Avança uma página
