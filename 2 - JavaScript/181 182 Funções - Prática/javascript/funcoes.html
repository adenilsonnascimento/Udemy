Em JavaScript, uma função é um bloco de código projetado para realizar uma tarefa específica. Ela pode ser definida uma vez e usada repetidamente em diferentes partes do código, o que ajuda a organizar e evitar repetição de código.

Como funciona:
Declaração de uma função: Para criar uma função, você usa a palavra-chave function, seguida por um nome para a função, parênteses que podem conter parâmetros (valores que a função pode receber), e um bloco de código dentro de chaves {} que define o que a função faz.

Exemplo:

javascript
Copiar código
function saudacao() {
console.log("Olá, bem-vindo!");
}
Chamada de uma função: Depois de declarar uma função, você pode chamá-la em qualquer lugar do código para executar o que foi definido dentro dela.

Exemplo:

javascript
Copiar código
saudacao(); // Saída: Olá, bem-vindo!
Parâmetros e Argumentos: Funções podem receber valores (parâmetros) que afetam seu comportamento. Esses valores são passados na chamada da função como argumentos.

Exemplo:

javascript
Copiar código
function saudacao(nome) {
console.log("Olá, " + nome + "!");
}

saudacao("Adenilson"); // Saída: Olá, Adenilson!
Retornando Valores: Funções podem retornar valores usando a palavra-chave return. Isso permite que a função envie um valor de volta para onde ela foi chamada.

Exemplo:

javascript
Copiar código
function soma(a, b) {
return a + b;
}

let resultado = soma(5, 3); // resultado agora é 8
console.log(resultado); // Saída: 8
Funções são uma parte essencial de JavaScript, permitindo modularização, reutilização de código e simplificação de tarefas complexas.
