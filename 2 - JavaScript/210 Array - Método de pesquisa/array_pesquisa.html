O código começa declarando o array lista_frutas com a variável let e atribuindo alguns elementos, como 'Banana', 'Maça', 'Morango' e 'Uva'.
O método indexOf() é usado para encontrar a posição de um item dentro do array:

  • Quando procuramos por 'Morango', o método retorna o índice do item no array (que é 2) e é impresso no console;
  • Quando procuramos por 'Abacaxi', que não existe no array, o método retorna -1, indicando que o item não foi encontrado.

O código ainda verifica se o item 'Banana' existe no array, e imprime a posição do elemento se ele for encontrado. Caso o elemento não exista, a mensagem 'Elemento não existe' é exibida.
Por fim, é importante notar que a pesquisa feita pelo método indexOf() é case sensitive, o que significa que o método diferencia letras maiúsculas de minúsculas. Por exemplo, a pesquisa por 'banana' (minúsculo) não retornaria a posição de 'Banana' (com B maiúsculo).
