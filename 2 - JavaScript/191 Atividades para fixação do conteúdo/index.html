Calculadora de Soma e Subtração

Este código é uma calculadora simples que realiza operações de soma e subtração, recebendo as entradas do usuário por meio de prompts:

  • let num1: recebe o primeiro número inserido pelo usuário.
  • let operacao: recebe o operador matemático escolhido pelo usuário ("+" para soma ou "-" para subtração).
  • let num2: recebe o segundo número inserido pelo usuário.
  • parseFloat: converte os valores inseridos em números de ponto flutuante para realizar operações matemáticas.
  • Bloco if-else: avalia o operador inserido e realiza a operação correspondente. Caso o operador seja inválido, uma mensagem de erro é exibida na página e o programa é encerrado com throw.
  • DOMContentLoaded: espera o carregamento completo da página para exibir o resultado em um elemento p criado dinamicamente.
