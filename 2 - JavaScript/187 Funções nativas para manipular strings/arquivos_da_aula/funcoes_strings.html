Vamos analisar e explicar cada parte do código JavaScript que demonstra algumas manipulações de strings utilizando métodos comuns.
1. Propriedade length

document.write('Jorge Sant Ana'.length)

A propriedade length retorna o número de caracteres de uma string.
No caso de 'Jorge Sant Ana', o valor retornado será 13 (incluindo os espaços em branco).
Saída esperada: 13

2. Método charAt()

document.write('Jorge Sant Ana'.charAt(7))
O método charAt() retorna o caractere de uma string com base no índice fornecido.
No exemplo, o índice 7 corresponde ao caractere 'S', que é o oitavo caractere da string (a contagem começa em 0).
Saída esperada:

3. Método indexOf()

var nome = 'Jorge Sant Ana'
document.write(nome.indexOf('g'))
O método indexOf() retorna o índice da primeira ocorrência de um caractere ou substring dentro da string.
No exemplo, o caractere 'g' aparece pela primeira vez no índice 2.
Saída esperada: 2

4. Método replace()

document.write(nome.replace('Sant Ana', 'Silva'))
O método replace() substitui uma parte específica da string por outra.
Aqui, 'Sant Ana' é substituído por 'Silva'.
Saída esperada: Jorge Silva

5. Método substr()

document.write(nome.substr(6,4))
O método substr() retorna uma parte da string a partir de um índice inicial, com base no número de caracteres a serem extraídos.
O primeiro argumento (6) é o índice inicial, e o segundo argumento (4) indica a quantidade de caracteres a serem retornados.
A partir do índice 6, os quatro caracteres são 'Sant'.
Saída esperada: Sant

6. Método toLowerCase() e toUpperCase()

document.write(nome.toLocaleLowerCase())
document.write(nome.toLocaleUpperCase())
O método toLowerCase() transforma todos os caracteres da string em letras minúsculas.
O método toUpperCase() transforma todos os caracteres em letras maiúsculas.
Saída esperada (para toLowerCase): jorge sant ana
Saída esperada (para toUpperCase): JORGE SANT ANA

7. Método trim()

document.write('-'+ nome.trim() + '-')
O método trim() remove os espaços em branco do início e do fim de uma string.
No exemplo, é usado para garantir que qualquer espaço no início ou fim seja removido, e a string é exibida entre dois hifens ('-').
Saída esperada: -Jorge Sant Ana-

Resumo dos métodos usados:
length: retorna o comprimento de uma string.
charAt(index): retorna o caractere no índice especificado.
indexOf(value): retorna o índice da primeira ocorrência do valor especificado.
replace(searchValue, newValue): substitui parte da string por outra.
substr(start, length): retorna uma parte da string, iniciando no índice especificado e com o comprimento indicado.
toLowerCase(): transforma a string em letras minúsculas.
toUpperCase(): transforma a string em letras maiúsculas.
trim(): remove os espaços em branco das extremidades da string.
