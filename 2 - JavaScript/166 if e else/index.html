O if e else são estruturas condicionais em JavaScript que permitem executar diferentes blocos de código com base em condições específicas. Elas são usadas para tomar decisões no código, verificando se uma condição é verdadeira ou falsa.

1. if (se)
A instrução if executa um bloco de código se a condição dentro dos parênteses for verdadeira (true).

Sintaxe:

javascript
Copiar código
if (condicao) {
// Bloco de código a ser executado se a condição for verdadeira
}

Exemplo:

javascript
Copiar código
let idade = 18;

if (idade >= 18) {
console.log("Você é maior de idade.");
}
Saída: Você é maior de idade.

2. else (senão)
A instrução else é usada para definir um bloco de código a ser executado se a condição do if for falsa.

Sintaxe:

javascript
Copiar código
if (condicao) {
// Bloco de código se a condição for verdadeira
} else {
// Bloco de código se a condição for falsa
}

Exemplo:
javascript
Copiar código
let idade = 16;
if (idade >= 18) {
console.log("Você é maior de idade.");
} else {
console.log("Você é menor de idade.");
}
Saída: Você é menor de idade.

3. else if
A instrução else if permite verificar condições adicionais se a primeira condição for falsa. Ela funciona como uma cadeia de verificações.

Sintaxe:

javascript
Copiar código
if (condicao1) {
// Bloco de código se condicao1 for verdadeira
} else if (condicao2) {
// Bloco de código se condicao2 for verdadeira
} else {
// Bloco de código se nenhuma das condições for verdadeira
}

Exemplo:

javascript
Copiar código
let hora = 15;

if (hora < 12) {
console.log("Bom dia!");
} else if (hora < 18) {
console.log("Boa tarde!");
} else {
console.log("Boa noite!");
}
Saída: Boa tarde!

4. Encadeamento de if...else
Você pode ter várias condições verificadas em sequência, e o JavaScript vai executar o primeiro bloco cujo teste for verdadeiro. Se nenhum teste for verdadeiro, o bloco else é executado.

Exemplo:

javascript
Copiar código
let nota = 75;

if (nota >= 90) {
console.log("Excelente!");
console.log("Bom trabalho!");
} else {
console.log("Precisa melhorar.");
}
Saída: Bom trabalho!

5. Uso de várias condições com operadores lógicos
Você pode combinar condições com os operadores lógicos, como && (AND) e || (OR), dentro do if.

Exemplo:

javascript
Copiar código
let idade = 20;
let possuiCarteira = true;

if (idade >= 18 && possuiCarteira) {
console.log("Pode dirigir.");
} else {
console.log("Não pode dirigir.");
}
Saída: Pode dirigir.

Resumo
if: Executa um bloco se a condição for verdadeira.
else: Executa um bloco se a condição do if for falsa.
else if: Permite verificar múltiplas condições.
