Este código demonstra o uso de laços for...in para iterar sobre arrays em JavaScript, além de explorar como arrays podem receber índices personalizados.

Funcionamento:
1. O array listaFamilia contém cinco nomes.
- O laço for...in itera sobre os índices do array, e para cada índice, exibe o índice e o valor correspondente no console.

2. O array listaConvidados é inicializado vazio, mas recebe valores associados a índices personalizados:
- 'a': Atribuído ao valor 'Adenilson'.
- 10: Atribuído ao valor 'Josyane'.
- 'zebra': Atribuído ao valor 'Jade'.
- -1: Atribuído ao valor 'Kamilly'.
- true: Atribuído ao valor 'Maria' (convertido para o índice 'true' como string).

3. Um segundo laço for...in itera sobre os índices personalizados do array listaConvidados, exibindo o índice e o valor no console.

Observação:
- Arrays em JavaScript podem usar índices numéricos ou não-numéricos, mas índices personalizados não são considerados no comprimento (length) do array.
- A exibição no console do array listaConvidados permite observar como ele armazena esses índices e valores.
