Este código demonstra o uso de eventos em JavaScript aplicados a um elemento `

` estilizado como um quadrado cinza. Os eventos são configurados diretamente no HTML usando atributos de manipulação de eventos:

  • O evento onclick está associado à função acao1(). Quando o quadrado é clicado, um alerta com a mensagem "Elemento foi clicado" é exibido.
  • O evento onmouseout está associado à função acao2(). Quando o cursor do mouse sai da área do quadrado, um alerta com a mensagem "O cursor do mouse deixou a região do elemento" é exibido.

Além disso, o código possui exemplos comentados de outros eventos do mouse, como:

  • onclick: Dispara quando o elemento é clicado.
  • ondblclick: Dispara quando o elemento é clicado duas vezes rapidamente.
  • onmouseup: Dispara quando o botão do mouse é liberado sobre o elemento.
  • onmouseover: Dispara quando o cursor do mouse entra na região do elemento.
  • onmouseout: Dispara quando o cursor do mouse sai da região do elemento.

As funções JavaScript são chamadas apenas quando os eventos correspondentes ocorrem, permitindo que interações dinâmicas sejam adicionadas ao site. Esses exemplos são úteis para entender como eventos podem ser utilizados para melhorar a experiência do usuário.
