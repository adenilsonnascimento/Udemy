Manipulando estilo

Explicações dos códigos

O código permite alterar dinamicamente o estilo de um quadrado na tela. A função `modificaEstilo` recebe três parâmetros: a cor de fundo, a largura e a altura do quadrado. Dentro dessa função, o método `document.getElementById('quadrado')` é utilizado para acessar o elemento com o ID "quadrado". Em seguida, são modificados três estilos do quadrado: - `backgroundColor` para mudar a cor de fundo. - `width` para definir a largura do quadrado. - `height` para definir a altura. Esses valores são passados quando o usuário clica nos botões, que chamam a função `modificaEstilo` com diferentes parâmetros para cada cor e tamanho.
