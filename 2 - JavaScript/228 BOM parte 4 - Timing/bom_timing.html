Explicação: O que é e quando usar o DOM Timing

O DOM Timing refere-se a métodos do JavaScript que permitem controlar eventos baseados no tempo, como executar uma função após um atraso (com setTimeout) ou repetir uma função em intervalos regulares (com setInterval). Esses métodos são amplamente utilizados para criar interações dinâmicas, como contagens regressivas, animações e atrasos.

No exemplo acima, usamos setTimeout para executar um comando único após um intervalo, e setInterval para executar um comando repetidamente a cada intervalo de tempo. O clearInterval é usado para parar o setInterval quando a condição desejada é atendida.

Explicando TODOS os códigos do DOM Timing

  • setTimeout: Executa uma função ou bloco de código após um atraso especificado em milissegundos. Exemplo comentado no código executaria document.write('Teste') após 3 segundos.
  • setInterval: Chama repetidamente uma função em intervalos regulares. No exemplo, escrevemos uma contagem regressiva começando de 5, com decremento a cada segundo.
  • clearInterval: Interrompe um intervalo criado com setInterval. No exemplo, ele é chamado quando a contagem regressiva chega a zero.
  • Segundo exemplo comentado de setInterval: Escreveria repetidamente "Teste" no documento a cada 2 segundos.
