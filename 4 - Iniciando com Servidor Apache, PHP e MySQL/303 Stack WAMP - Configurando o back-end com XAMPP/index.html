Stack WAMP - Configurando o back-end com XAMPP

A Stack WAMP (Windows, Apache, MySQL, PHP) é uma das configurações de servidor mais populares para desenvolvimento web em ambientes Windows. Para facilitar a configuração, muitos desenvolvedores utilizam o XAMPP, um pacote de software que inclui todos os componentes necessários para rodar um servidor local.

1. O que é o XAMPP?

XAMPP é uma distribuição Apache totalmente gratuita e fácil de instalar, que contém MariaDB (um fork do MySQL), PHP e Perl. Ele foi projetado para tornar o desenvolvimento web mais simples e é multiplataforma, funcionando em Windows, Linux e macOS.

2. Instalando o XAMPP

Para instalar o XAMPP no Windows, siga estes passos:

  1. Acesse o site oficial do XAMPP em Apache Friends.
  2. Baixe o instalador do XAMPP para Windows.
  3. Execute o instalador e siga as instruções na tela.
  4. Durante a instalação, você pode escolher os componentes que deseja instalar. Certifique-se de que Apache, MySQL e PHP estejam selecionados.
  5. Conclua a instalação e inicie o XAMPP Control Panel.

3. Configurando o Servidor Apache

Após instalar o XAMPP, você pode configurar o servidor Apache para rodar suas aplicações web:

  • No XAMPP Control Panel, clique em "Start" ao lado de Apache para iniciar o servidor.
  • Para verificar se o Apache está funcionando, abra o navegador e acesse http://localhost. Se tudo estiver configurado corretamente, você verá a página inicial do XAMPP.

4. Configurando o Banco de Dados com MariaDB

MariaDB é usado como o sistema de gerenciamento de banco de dados no XAMPP. Para configurá-lo:

  • No XAMPP Control Panel, clique em "Start" ao lado de MySQL para iniciar o servidor de banco de dados.
  • Acesse http://localhost/phpmyadmin no navegador para abrir o phpMyAdmin, uma interface gráfica para gerenciar bancos de dados MariaDB.
  • Você pode criar novos bancos de dados, tabelas, inserir dados e executar consultas SQL diretamente no phpMyAdmin.

5. Executando Aplicações PHP

Para executar suas aplicações PHP, salve seus arquivos na pasta htdocs dentro do diretório onde o XAMPP foi instalado (normalmente C:\xampp\htdocs).

Acesse seus scripts PHP através do navegador, digitando a URL http://localhost/nome_do_seu_arquivo.php.

6. Vantagens de Usar XAMPP

O XAMPP oferece várias vantagens para desenvolvedores, incluindo:

  • Facilidade de instalação e configuração.
  • Inclui todos os componentes necessários para o desenvolvimento web em um único pacote.
  • Suporte a múltiplas plataformas.
  • Ferramentas adicionais como o phpMyAdmin para gerenciar bancos de dados.

Para mais informações sobre como usar e configurar o XAMPP, visite o site oficial em Apache Friends.
