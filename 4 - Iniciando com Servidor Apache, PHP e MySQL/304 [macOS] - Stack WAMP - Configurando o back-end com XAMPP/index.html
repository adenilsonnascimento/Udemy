[macOS] - Stack WAMP - Configurando o back-end com XAMPP

Embora a sigla WAMP (Windows, Apache, MySQL, PHP) seja originalmente direcionada para o sistema operacional Windows, os mesmos componentes podem ser utilizados no macOS com a ajuda do XAMPP. O XAMPP oferece uma maneira fácil de configurar um ambiente de desenvolvimento web no macOS.

1. O que é o XAMPP?

XAMPP é uma distribuição Apache que inclui MariaDB, PHP e Perl, projetada para facilitar o desenvolvimento web. O XAMPP é multiplataforma, o que significa que você pode usá-lo tanto no Windows quanto no macOS.

2. Instalando o XAMPP no macOS

Para instalar o XAMPP no macOS, siga estas etapas:

  1. Visite o site oficial do XAMPP em Apache Friends.
  2. Baixe a versão mais recente do XAMPP para macOS.
  3. Após o download, abra o arquivo .dmg e arraste o XAMPP para a pasta "Aplicativos".
  4. Abra o XAMPP a partir da pasta "Aplicativos".

3. Configurando o Servidor Apache

Depois de instalar o XAMPP, você pode configurar o servidor Apache no macOS:

  • Abra o XAMPP e clique em "Start" ao lado de Apache para iniciar o servidor.
  • Para verificar se o Apache está funcionando, abra o navegador e acesse http://localhost. Se tudo estiver configurado corretamente, você verá a página inicial do XAMPP.

4. Configurando o Banco de Dados com MariaDB

O XAMPP usa MariaDB como sistema de gerenciamento de banco de dados. Para configurá-lo:

  • Na interface do XAMPP, clique em "Start" ao lado de MySQL para iniciar o banco de dados MariaDB.
  • Acesse http://localhost/phpmyadmin no navegador para abrir o phpMyAdmin, onde você pode gerenciar bancos de dados MariaDB.

5. Executando Aplicações PHP

Para rodar seus scripts PHP:

  • Salve seus arquivos PHP na pasta /Applications/XAMPP/htdocs.
  • Acesse seus scripts PHP digitando http://localhost/nome_do_seu_arquivo.php no navegador.

6. Vantagens de Usar XAMPP no macOS

O uso do XAMPP no macOS oferece várias vantagens:

  • Configuração fácil e rápida de um ambiente de desenvolvimento local.
  • Inclui todos os componentes essenciais como Apache, MariaDB e PHP.
  • Interface gráfica intuitiva para gerenciar o servidor e os serviços.

Para mais informações sobre como usar e configurar o XAMPP no macOS, visite o site oficial em Apache Friends.
