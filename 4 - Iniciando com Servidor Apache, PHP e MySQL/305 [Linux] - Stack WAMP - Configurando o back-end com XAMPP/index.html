[Linux] - Stack WAMP - Configurando o back-end com XAMPP

Embora a sigla WAMP (Windows, Apache, MySQL, PHP) se refira originalmente ao ambiente Windows, os componentes desta stack também podem ser utilizados no Linux com a ajuda do XAMPP. O XAMPP oferece uma maneira simples e eficiente de configurar um ambiente de desenvolvimento web no Linux.

1. O que é o XAMPP?

XAMPP é uma distribuição que inclui Apache, MariaDB, PHP e Perl, projetada para facilitar o desenvolvimento web local. Ele é multiplataforma, tornando-se uma escolha popular para desenvolvedores que trabalham em Linux.

2. Instalando o XAMPP no Linux

Para instalar o XAMPP no Linux, siga os passos abaixo:

  1. Visite o site oficial do XAMPP em Apache Friends.
  2. Baixe o instalador do XAMPP para Linux.
  3. Abra o terminal e navegue até o diretório onde o instalador foi baixado.
  4. Execute o comando sudo chmod +x xampp-linux-*-installer.run para tornar o instalador executável.
  5. Execute o instalador com o comando sudo ./xampp-linux-*-installer.run e siga as instruções na tela.

3. Configurando o Servidor Apache

Após instalar o XAMPP, você pode configurar o servidor Apache no Linux:

  • Abra o terminal e inicie o XAMPP com o comando sudo /opt/lampp/lampp start.
  • Para verificar se o Apache está funcionando, abra o navegador e acesse http://localhost. Se estiver tudo certo, a página inicial do XAMPP será exibida.

4. Configurando o Banco de Dados com MariaDB

O XAMPP usa MariaDB como sistema de gerenciamento de banco de dados. Para configurá-lo no Linux:

  • O MariaDB é iniciado automaticamente com o XAMPP. Para verificar, acesse http://localhost/phpmyadmin no navegador.
  • O phpMyAdmin fornece uma interface gráfica para gerenciar bancos de dados MariaDB.

5. Executando Aplicações PHP

Para executar seus scripts PHP no Linux:

  • Salve seus arquivos PHP no diretório /opt/lampp/htdocs.
  • Acesse seus scripts PHP digitando http://localhost/nome_do_seu_arquivo.php no navegador.

6. Vantagens de Usar XAMPP no Linux

O XAMPP oferece diversas vantagens para os desenvolvedores Linux:

  • Instalação simples e rápida de um ambiente de desenvolvimento completo.
  • Inclui todos os componentes necessários para o desenvolvimento web em um único pacote.
  • Suporte para diferentes distribuições Linux.

Para mais informações sobre como usar e configurar o XAMPP no Linux, visite o site oficial em Apache Friends.
