Stack WAMP - Conhecendo o XAMPP

O XAMPP é uma ferramenta popular que fornece uma stack de desenvolvimento web no formato WAMP (Windows, Apache, MySQL, PHP), permitindo que você crie e teste sites localmente. Ele é multiplataforma e pode ser instalado em sistemas operacionais como Windows, macOS e Linux.

1. O que é o XAMPP?

XAMPP é uma distribuição de código aberto que inclui Apache, MariaDB, PHP e Perl, projetada para facilitar o desenvolvimento web. Ele oferece um ambiente completo para criar e testar sites em PHP sem a necessidade de se preocupar com a instalação e configuração de servidores e bancos de dados manualmente.

2. Como usar o XAMPP

Após instalar o XAMPP, você pode iniciar e parar os serviços Apache e MySQL diretamente pela interface gráfica do XAMPP.

  • Inicie o XAMPP e abra a interface gráfica.
  • Clique em "Start" ao lado de "Apache" para iniciar o servidor web.
  • Clique em "Start" ao lado de "MySQL" para iniciar o banco de dados MariaDB.
  • Quando ambos os serviços estiverem em execução, você verá as palavras "Running" ao lado deles.

3. O que o XAMPP faz?

O XAMPP oferece os seguintes componentes principais:

  • Apache: Servidor web que processa as requisições HTTP e exibe suas páginas no navegador.
  • MariaDB: Sistema de gerenciamento de banco de dados (relacional) usado para armazenar dados das aplicações web.
  • PHP: Linguagem de programação usada para criar sites dinâmicos e interativos.
  • Perl: Linguagem de programação adicional que pode ser usada para tarefas de automação e processamento de texto.

4. Portas do XAMPP

O XAMPP, por padrão, utiliza as seguintes portas:

  • Apache: Porta 80 para HTTP (padrão), porta 443 para HTTPS.
  • MySQL: Porta 3306 para o banco de dados MariaDB.

Se você já tiver outro servidor web em execução (como o IIS ou Nginx), pode ser necessário mudar a porta padrão do Apache. Para fazer isso, siga os passos abaixo:

  1. Abra o XAMPP e clique em "Config" ao lado de Apache.
  2. Escolha "httpd.conf" para editar o arquivo de configuração do Apache.
  3. Procure por Listen 80 e altere o número 80 para outra porta, como 8080.
  4. Salve o arquivo e reinicie o Apache.

Agora, ao acessar o servidor local, você precisará digitar a nova porta na URL, como http://localhost:8080.

5. Criando o Primeiro Arquivo PHP

Agora que o XAMPP está funcionando corretamente, você pode começar a criar seu primeiro arquivo PHP. Siga os passos abaixo:

  • Abra o diretório /htdocs dentro da pasta onde o XAMPP foi instalado. No Windows, isso geralmente fica em C:\xampp\htdocs. No macOS e Linux, o caminho é /opt/lampp/htdocs.
  • Crie um novo arquivo de texto e salve-o como index.php no diretório /htdocs.
  • Abra o arquivo index.php em um editor de texto e adicione o seguinte código:
    •             <?php
                echo "Olá, mundo! Meu primeiro arquivo PHP!";
            ?>
  • Salve o arquivo e abra o navegador.
  • Digite http://localhost/index.php na barra de endereços. Você deve ver a mensagem "Olá, mundo! Meu primeiro arquivo PHP!" exibida na tela.

6. Conclusão

O XAMPP facilita a criação e o teste de sites dinâmicos, oferecendo todos os componentes necessários (Apache, MySQL, PHP e Perl) em uma única instalação. Ao configurar corretamente as portas e criar seus primeiros arquivos PHP, você estará pronto para desenvolver e testar suas aplicações web localmente antes de colocá-las em produção.

Para mais informações sobre o XAMPP e como configurá-lo, acesse o site oficial: Apache Friends.
