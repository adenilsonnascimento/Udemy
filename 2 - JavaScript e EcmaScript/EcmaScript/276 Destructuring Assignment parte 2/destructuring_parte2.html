Exemplo de Destructuring com Objetos

O destructuring é uma forma concisa de extrair dados de objetos ou arrays e atribuí-los a variáveis. Neste exemplo, usamos destructuring para acessar as propriedades de um objeto aninhado. Através da sintaxe de { detalhes: { fabricante: f, modelo = 'Não informado' } }, conseguimos acessar diretamente as propriedades do objeto detalhes, atribuindo os valores às variáveis f e modelo, com um valor padrão para modelo.
