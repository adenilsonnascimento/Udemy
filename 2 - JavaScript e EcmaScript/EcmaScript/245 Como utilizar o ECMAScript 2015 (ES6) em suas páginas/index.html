Como utilizar o EcmaScript 2015

O EcmaScript 2015 (ES6) trouxe diversas melhorias para o JavaScript, tornando o código mais legível e eficiente. A seguir, abordaremos algumas das principais características e como utilizá-las.

1. Declaração de variáveis com let e const

O ES6 introduziu os blocos de escopo com as palavras-chave let e const, que são alternativas ao tradicional var.


// Exemplo de uso do let
let nome = 'João';
nome = 'Maria';  // Variável pode ser reatribuída

// Exemplo de uso do const
const idade = 30;
// idade = 31;  // Erro! 'const' não permite reatribuição

2. Arrow Functions

As arrow functions permitem uma forma mais concisa de declarar funções, além de manter o this do contexto onde a função foi criada.


// Exemplo de arrow function
const somar = (a, b) => a + b;

console.log(somar(2, 3));  // Resultado: 5

3. Classes

O ES6 trouxe a sintaxe de classes para o JavaScript, facilitando a criação e organização de objetos e métodos.


// Exemplo de classe no ES6
class Pessoa {
    constructor(nome, idade) {
        this.nome = nome;
        this.idade = idade;
    }

    apresentar() {
        console.log(`Meu nome é ${this.nome} e tenho ${this.idade} anos.`);
    }
}

const pessoa1 = new Pessoa('Carlos', 28);
pessoa1.apresentar();  // Resultado: "Meu nome é Carlos e tenho 28 anos."

4. Promises

O ES6 introduziu as Promises, que ajudam a lidar com operações assíncronas de forma mais clara e legível.


// Exemplo de Promise
const minhaPromise = new Promise((resolve, reject) => {
    let sucesso = true;
    
    if(sucesso) {
        resolve("Operação bem-sucedida!");
    } else {
        reject("Houve um erro.");
    }
});

minhaPromise
    .then(result => console.log(result))  // Resultado: "Operação bem-sucedida!"
    .catch(error => console.log(error));  // Caso ocorra erro

5. Módulos (Importação e Exportação)

O ES6 introduziu a capacidade de dividir código em módulos, facilitando a organização do código e a reutilização.


// Exemplo de exportação (arquivo pessoa.js)
export class Pessoa {
    constructor(nome, idade) {
        this.nome = nome;
        this.idade = idade;
    }
}

// Exemplo de importação (arquivo principal.js)
import { Pessoa } from './pessoa.js';

const pessoa1 = new Pessoa('Ana', 25);
console.log(pessoa1.nome);  // Resultado: "Ana"
